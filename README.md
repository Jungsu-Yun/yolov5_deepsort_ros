# YoloV5 DeepSORT
본 프로젝트는 ROS, yolov5를 활용하여 객체를 탐지해내고, Deepsort 알고리즘을 활용하여 단일 객체를 분류 및 추적하는 기능을 ROS 패키지로 구현하였습니다.
본 프로젝트의 원 소스코드는 아래와 같습니다.

[fcakyon/yolov5-pip](https://github.com/fcakyon/yolov5-pip.git)
